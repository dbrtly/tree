# Tech Stack

## Programming Language

- **Zig:** The core programming language for the project, known for its focus on performance, safety, and control.

## Build System & Dependencies

- **Zig Build System:** Utilizes Zig's native build system for project compilation, testing, and other development tasks.
- **zlinter:** A Zig linter used for maintaining code quality and style.
- **kcov:** A code coverage analysis tool used for generating code coverage reports.
